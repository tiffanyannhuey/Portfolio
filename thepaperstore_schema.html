The Paper Store Google Structured Data with YotPo

The Paper Store requested to add schema data to the storefront, category, and product pages as requested by their SEO team.

Category page structure data was built in carousel format: https://developers.google.com/search/docs/guides/mark-up-listings

For the product schema data, I noticed that their Yotpo integration includes a separate product schema data out of the package for products that have reviews, which could not be modified. I removed this and added a manual call to Yotpo API so that I can insert their aggregate review data along with other product schema data for name, description, brand, and offers.
