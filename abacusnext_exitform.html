AbacusNext Exit Intent Webform Campaign

The client (AbacusNext) wanted more ways to generate leads for their Sales pipeline. To get more visibility on current promotions and to generate an additional pageview, I proposed creating an exit intent campaign on Abacusnext.com. The exit modal form was built using the Drupal Bounce Convert module and Bootstrap CDN. The modal window can be customized to appear on certain pages and multiple campaigns can be run concurrently. Once the visitor views the form, a browser cookie can be set so that the modal does not appear again for a set period of time.

Note: The exit intent modal popup is fully functional but is currently on the dev site, pending final approval for campaign ideas/design. Some ideas that can be used for the campaign are:

  • Customer Satisfaction Survey - Did you find what you were looking for?
  • Campaign on paid landing pages to add additional offer to visitor upon exit
  • Newsletter subscription emails
